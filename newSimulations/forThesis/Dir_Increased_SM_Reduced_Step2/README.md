# Dir_Increased_SM_Reduced_Step2

## 実験目的

Director の管理スパンを増加させ、SeniorManager の管理スパンを削減した場合の負荷変化を観察する。

## パラメータ設定

### 管理スパン（子ノード数）

| 役職 | 子ノード範囲 | 平均 | InterviewBasedからの変化 |
|------|-------------|------|-------------------------|
| CXO | [15, 20] | 17.5 | 変更なし |
| Director | [14, 19] | 16.5 | +32% (元: [10,15]) |
| SeniorManager | [6, 9] | 7.5 | -40% (元: [10,15]) |
| Manager | [8, 11] | 9.5 | +27% (元: [5,10]) |
| Player | 0 | 0 | 変更なし |

※ Manager→Player のスパン調整は総ノード数を維持するため

### シミュレーション条件

- TIME_STEPS: 2000
- 乱数シード: 42
- タスク生成: Poisson分布 (λ=2)

## 結果概要

- 総ノード数: 25,129
- InterviewBased (25,731) との差: -602 ノード

### 役職別ノード数・負荷

| 役職 | ノード数 | 平均タスク数 | 平均SimLoad |
|------|---------|-------------|-------------|
| CXO | 1 | 19,475 | 44,920 |
| Director | 20 | 20,130 | 67,232 |
| SeniorManager | 318 | 27,331 | 80,979 |
| Manager | 2,348 | 69,589 | 68,762 |
| Player | 22,442 | 6,612 | 6,778 |
