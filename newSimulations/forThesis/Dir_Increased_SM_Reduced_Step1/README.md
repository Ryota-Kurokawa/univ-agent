# Dir_Increased_SM_Reduced_Step1

## 実験目的

Director の管理スパンを増加させ、SeniorManager の管理スパンを削減した場合の負荷変化を観察する。

## パラメータ設定

### 管理スパン（子ノード数）

| 役職 | 子ノード範囲 | 平均 | InterviewBasedからの変化 |
|------|-------------|------|-------------------------|
| CXO | [15, 20] | 17.5 | 変更なし |
| Director | [12, 17] | 14.5 | +16% (元: [10,15]) |
| SeniorManager | [8, 11] | 9.5 | -24% (元: [10,15]) |
| Manager | [7, 10] | 8.5 | +13% (元: [5,10]) |
| Player | 0 | 0 | 変更なし |

※ Manager→Player のスパン調整は総ノード数を維持するため

### シミュレーション条件

- TIME_STEPS: 2000
- 乱数シード: 42
- タスク生成: Poisson分布 (λ=2)

## 結果概要

- 総ノード数: 25,211
- InterviewBased (25,731) との差: -520 ノード

### 役職別ノード数・負荷

| 役職 | ノード数 | 平均タスク数 | 平均SimLoad |
|------|---------|-------------|-------------|
| CXO | 1 | 19,589 | 45,380 |
| Director | 20 | 20,174 | 69,752 |
| SeniorManager | 278 | 30,699 | 86,139 |
| Manager | 2,611 | 63,225 | 66,789 |
| Player | 22,301 | 6,650 | 7,165 |
