# Dir_Increased_SM_Reduced_Step3

## 実験目的

Director の管理スパンを増加させ、SeniorManager の管理スパンを削減した場合の負荷変化を観察する。

## パラメータ設定

### 管理スパン（子ノード数）

| 役職 | 子ノード範囲 | 平均 | InterviewBasedからの変化 |
|------|-------------|------|-------------------------|
| CXO | [15, 20] | 17.5 | 変更なし |
| Director | [16, 21] | 18.5 | +48% (元: [10,15]) |
| SeniorManager | [5, 7] | 6.0 | -52% (元: [10,15]) |
| Manager | [9, 12] | 10.5 | +40% (元: [5,10]) |
| Player | 0 | 0 | 変更なし |

※ Manager→Player のスパン調整は総ノード数を維持するため

### シミュレーション条件

- TIME_STEPS: 2000
- 乱数シード: 42
- タスク生成: Poisson分布 (λ=2)

## 結果概要

- 総ノード数: 25,140
- InterviewBased (25,731) との差: -591 ノード

### 役職別ノード数・負荷

| 役職 | ノード数 | 平均タスク数 | 平均SimLoad |
|------|---------|-------------|-------------|
| CXO | 1 | 19,682 | 44,040 |
| Director | 20 | 20,452 | 67,778 |
| SeniorManager | 358 | 25,025 | 77,134 |
| Manager | 2,142 | 76,593 | 70,986 |
| Player | 22,619 | 6,646 | 6,488 |
